## About
This game was created as part of a MM/AN 40A class Introduction to Game Design at Berkeley City College, Spring 2026. The name of this game is called Run 'n' Jump. I wanted to add a nightsky because I wanted the game to have a good background, even if its my first project as a beginner. I also added elements of flying bombs and a star for the game to look a bit fun for people to play. The spikes came from insipration from the game Geometry Dash, a game I find myself playing on my freetime. This game is for people to see what students like me and my peers work on for our first game, and wanting to show our hard work for this year. This game is a short and simple platform game.
